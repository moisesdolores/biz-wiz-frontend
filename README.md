## Usage

To run on local machine, run command from within the Biz-Wiz-Frontend directory:

```sh
- npm start
```

## Requirements

- React
- React Router Dom
- Material-UI
- Node 0.10.x

## Development

### Installing Dependencies

From within the Biz-Wiz-Frontend directory:

```sh
- npm install react-router-dom 
- npm install axios
- npm install @material-ui/core
```
